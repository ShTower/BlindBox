## 运行说明
1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动应用：
   ```bash
   npm start
   ```

3. 访问应用：
   - 用户界面: http://localhost:3000
   - 管理后台: http://localhost:3000/admin


## 技术栈
- 后端：Node.js + Express
- 前端：React + Vite
- 数据库：SQLite
